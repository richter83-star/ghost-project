# DRACANUS AI Brand Guide

## Brand Identity
**Dark, Metallic, Tech-Forward AI Brand**

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#1a1a1a` | Main background |
| **Secondary** | `#2d2d2d` | Cards, panels |
| **Accent** | `#ffffff` | Headings, highlights |
| **Metallic** | `#4a4a4a` | Borders, buttons |
| **Dark** | `#0a0a0a` | Headers, footers |
| **Text** | `#e0e0e0` | Body text |
| **Text Muted** | `#999999` | Secondary text |

## Typography

- **Headings**: Inter (bold, 700 weight)
- **Body**: Inter (regular, 400 weight)
- **Code/Mono**: SF Mono, Monaco, Courier New

## Design Principles

1. **Sharp & Angular** - Minimal border radius (4px)
2. **Metallic Accents** - Subtle gradients and highlights
3. **High Contrast** - White on dark for readability
4. **Tech Aesthetic** - Clean, modern, powerful
5. **3D Effects** - Subtle shadows and embossed text

## Shopify Theme Settings

### Colors
- Background: `#1a1a1a`
- Text: `#e0e0e0`
- Accent: `#ffffff`
- Buttons: `#4a4a4a` with white text

### Typography
- Heading font: Inter
- Body font: Inter
- Font weight: Bold for headings (700), Regular for body (400)

### Logo
- Upload DRACANUS AI logo to Header section
- Recommended size: 200px width
- Format: PNG with transparent background

## CSS Classes

The theme includes these custom classes:
- `.dracanus-metallic` - Metallic gradient background
- `.dracanus-accent` - White accent color
- `.section-divider` - Metallic divider line

## Implementation

Run: `node apply-dracanus-theme.mjs`

This will:
1. Apply custom CSS to your theme
2. Set up dark metallic styling
3. Configure buttons, cards, and navigation
4. Add hover effects and transitions
